Airplane Crashes Analysis 1908–2009

  • Data Analytics Immersive Program
  • General Assembly, Bahrain
  • Team: Murad Afzal, Jamal AlTaitoon, Azeem Afzal & Jubran AlTaitoon
  • Tools: Jupyter Notebook, Python – Pandas, Numpy, Seaborn and Plotly

This project provides a comprehensive analysis of global airplane crash data from 1908 to 2009. We conducted rigorous data cleaning, explored variable distributions and correlations, and formulated targeted research questions to uncover key insights. Through detailed analysis and compelling visualizations, we aim to reveal historical trends, patterns, and potential contributing factors related to aviation accidents.

Data Source: Airplane Crashes Since 1908 - Kaggle
